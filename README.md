# The Governer's Gambit Cipher Utils

This repository is part of **"The Governer's Gambit"** challenge. It contains a simple command-line tool for working with ciphers.

## Important Notice

The command-line tool in this repository **currently contains errors/bugs** and is **not ready for direct use**.  
Before getting started, you should:

- **Read through the code**
- **Identify and fix the errors**

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd cipher-utils
   ```

2. **Inspect and fix the code**:
   - Open the project in your editor.
   - Review the command-line interface and cipher logic.
   - Fix any exceptions, incorrect behavior, or incomplete features.

3. **Run the tool** (after fixing issues):
   ```bash
   python main.py
   ```
